# 🎲 보드게임

{% hint style="danger" %}
**보드게임 판에 삼지창을 날리지 마세요**

삼지창이 증발할 수 있으며, 이에 대한 복구가 불가능합니다.
{% endhint %}

{% hint style="warning" %}
**영역보호를 적용받지 않는 아이템입니다**

원하는 장소에서 설치할 수 있으며, 공용 장소에서 제거를 안할 시 경고의 대상이 될 수 있습니다.
{% endhint %}
