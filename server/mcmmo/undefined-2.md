# 약초학

약초학-Herbalism



약초학이란 식물을 캤을 때 올라가는 스킬입니다.



스킬

\-----------------------------------------------------------------------------------------------------

**이름:** Double Drops (2배 드롭)

**종류:** <mark style="color:blue;">패시브 스킬</mark>

**작동방법:** 플레이어가 식물과 작물을 때, 자동적으로 작동합니다.

**설명**: 레벨에 따라 자연적으로 생성된 광을 파괴할시 광물이 2배로 나옵니다.
