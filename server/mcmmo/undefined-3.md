# 채광

채광-Mining



채광란 물고기가 찌를 무는 속도를 줄여주고, 유용한 전리품을 끌어올릴 기회를 늘립니다. 또한 몬스터를 죽일 필요없이 전리품을 얻을 수 있도록 합니다.
