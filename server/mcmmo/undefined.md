# 🔨 발굴

발굴-Excavation

<mark style="color:green;">**스킬**</mark>

\-----------------------------------------------------------------------------------------------------

**이름:** Giga Drill Breaker (기가 드릴 브레이커)

**종류:** <mark style="color:red;">액티브 스킬</mark>

**작동방법:** 삽을 든 상태에서 우클릭

**설명**: 기가 드릴 브레이커는 일시적으로 삽에 효율레벨을 5씩 증가시킵니다.
