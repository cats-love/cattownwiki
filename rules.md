# 🔔 규칙

{% hint style="info" %}
**서버 내 규칙은 수정 및 보완될 수 있습니다**

규칙을 읽지 않아 생기는 불이익을 막기 위해 꼭 읽어주세요!
{% endhint %}

{% hint style="warning" %}
**서버는 5회 경고 제도를 운영중입니다.**

* **경고 횟수는 디스코드 신고결과에 기록**되며, <mark style="color:red;">**경고 횟수에 따른 기간밴**</mark>을 부여하고 있습니다.
* **5점 이상 경고 누적 후 언밴을 위해서는 반성문 작성이 필요**하며, <mark style="color:green;">**디스코드 해명게시판으로 반성문을 작성**</mark>할 수 있습니다.
* <mark style="color:red;">**해명게시판 작성으로 인한 언밴 후 경고누적으로 인한 밴 시에는 재 해명 불가**</mark>합니다.
{% endhint %}

### 『 1. 채팅 및 표현물 』

{% hint style="warning" %}
표현물은 <mark style="color:green;">**표지판, 책 등 종류를 불문**</mark>하며, 위반시 **뮤트** 또는 <mark style="color:red;">**1\~5점의 경고**</mark>를 부여합니다

* **서버 내 / 디스코드의 모든 게시판에서 동일한 효력**을 갖습니다.
* **부적절한 닉네임, 스킨, 마을명, 부동산명 등을 모두 포함**합니다.
{% endhint %}

* 지속적 **도배, 어그로 , 구걸, 욕설, 비방, 인종차별, 종교, 정치에 관한 발언 및 표현**을 금합니다.
* ㄴ채팅 내에서 확인되는 표현시에는 뮤트 10분 / 확성기로 확인되는 경우에는 예외없이 <mark style="color:red;">**경고**</mark>를 부여합니다.
* **특정 및 다수 유저에게 성적 수치심이나 불쾌감을 주는 발언 및 표현** 등을 금합니다.
* **분쟁 사항은 당사자 간 합의를 통해 처리하며,** 필요 시 관리자의 중재가 가능합니다.
* 공공장소에서의 **반말 및 과도한 tpa 요청**을 금합니다.
* **타게임 언급시 구두경고**가 부여되며, **지속적인 언급시에** <mark style="color:red;">**경고**</mark>를 부여합니다.
* **타게임 언급 / 플레이의 경우**에는 <mark style="color:green;">**디스코드의 타게임 통화방을 이용**</mark>하여야 합니다.
* **타서버 언급 및 타게임 유도시에는** <mark style="color:red;">**경고 없이 벤**</mark>처리로 부여됩니다.
* **섭접 발언, 서버 비하, 운영진 비하는 **<mark style="color:red;">**3\~5점의 경고**</mark>가 부여됩니다.
* **거래 및 구인구직** 등은 <mark style="color:green;">**확성기\[/확성기 (할말)]를 사용**</mark>하여야 합니다.\
  ㄴ확성기를 거래 및 구인구직 등 홍보 목적 을 제외한 다른 목적으로 사용 시 \
  &#x20;   <mark style="color:red;">**경고**</mark>를 부여합니다.
* **/e 명령어**에 있는 **내용** 사용 시 <mark style="color:red;">**경고**</mark>를 부여합니다.
* **한글 닉네임**은 <mark style="color:red;">**중복이 불가능**</mark>하며, <mark style="color:green;">**칭호**</mark>는 <mark style="color:green;"><mark style="color:red;">**서로 합의**<mark style="color:red;"></mark>하에 <mark style="color:green;">**중복으로 설정**</mark> 가능합니다\
  ㄴ <mark style="color:green;">**한글닉네임 변경 시**</mark> 닉네임이 <mark style="color:green;">**중복**</mark>인지 확인 부탁드리며, \
  &#x20;    <mark style="color:red;">**악의적인 변경 시 경고**</mark>가 부여됩니다.\
  ㄴ **한글 닉네임**은 <mark style="color:green;">**먼저 닉네임을 변경하신 분**</mark>께서 <mark style="color:green;">**우선권**</mark>을 가지고 있으며,\
  &#x20;    <mark style="color:green;">**/영닉 한글닉네임**</mark> 검색 시 <mark style="color:green;">**닉네임이 있는지 없는지 확인**</mark> 가능합니다

### 『2. 핵/프로그램/변조/버그악용/테러』

{% hint style="danger" %}
위반시 **해당 기능을 활용하여 발생한 아이템 몰수**와 <mark style="color:red;">**3\~5점의 경고**</mark>를 부여합니다.
{% endhint %}

* 플레이에 영향을 주는 **변조 프로그램과 모드, 핵, 엑스레이 등**을 금합니다. **\[해명불가]**
* 형평성 문제가 발생하는 **모든 형태의 매크로 및 외부 프로그램**을 금합니다.
* **지역보호가 풀린 땅은 서버 소유이며, 빈 부지의 아이템/건축물을 가져가는 경우에는 약탈로 구분**합니다.
* 부동산이 존재하는 **건축물에 피해를 입히거나, 약탈 및 테러를 금합**니다.
* **게임 시스템 상 버그 악용**을 금합니다.
* 서버에서 **허용되는 모드**는 <mark style="color:green;">**" 한글채팅, Minimap, Optifine,Iris, Sodium"**</mark> 이며, 그 외의 모드는\
  디스코드 질문하기를 통하여 <mark style="color:red;">**허락**</mark> 후 사용하시길 바랍니다.

### 『3. 부계정』

* **모든 부계정은 금지합니다.** <mark style="color:green;">**(가족계정의 경우, 관리자 개인 DM으로 닉네임과 함께 문의)**</mark>

### 『4. 특정 플레이어에 의한 지속적 피해 발생』

{% hint style="warning" %}
관리자는 **논의를 통해 적절한 제재**를 가할 수 있으며, <mark style="color:red;">**1\~5점의 경고**</mark>를 부여합니다.
{% endhint %}

* **3인 이상의 플레이어**로부터 피해 사실이 신고된 경우
* 관리진 회의를 통해 **해당 플레이어가 다수에게 직간접적 피해를 끼치는 것이 명백하다고 판단**될 경우
* 현행 **규칙의 미비로 인해 처벌하기 힘든 피해 행위**를 발생시켰을 경우
* **고의적,지속적으로 타 유저의 플레이를 방해**하는 경우

### 『5. 레드스톤 기계』

{% hint style="danger" %}
<mark style="color:green;">**반자동 및 자동농사는 벨런스 상 서버에서 금지**</mark>하고 있으며, <mark style="color:red;">**1\~3점의 경고**</mark>를 부여합니다.
{% endhint %}

* 레드스톤 장치는 로그가 기록되며, 불시에 관리자가 해당 장치를 확인 할 수 있습니다.
* 형평성 문제를 발생시키는 **완전한 형태의 자동농사는 반자동으로 판단**합니다. <mark style="color:green;">**(레드스톤 및 호퍼를 사용하는 농장은 금합니다.)**</mark>
* **무한 회로의 경우 서버에서 자동으로 제거**하며, 서버의 심각한 지장을 초래한다고 판단 시에 무통보 제거될 수 있습니다.
* **해당 장치가 서버 및 구성원에 피해를 준다고 판단되는 경우 수정 요청이 가능**하며, 이에 불응 시 관리자의 직권으로 제거가 가능합니다.

### 『6. 엔티티 수』

{% hint style="warning" %}
**엔티티 수는 좌표 X/Y/Z 한청크 로 확인합니다.**

<mark style="color:green;">**\[ 청크 확인 방법 : F3 + G ]**</mark>
{% endhint %}

**주민 또는 동물이 밀집되어 있으면 시스템에서 자동으로 제거하며, 관리진이 보기에 부적절하다고 판단되면 삭제 처리 될 수 있습니다.**

**⚙ 호퍼**

* **청크당 10개까지 설치가 가능하며,** <mark style="color:green;">**10개를 초과할 경우 설치가 불가능**</mark>합니다.
* **레드스톤 회로 및 달걀, 인갑등의 수집을 허용하나, 공장 및 농장 목적으로 사용할 수 없습니다.**

🐓 **동물**

* **청크당 5마리까지 허용**되며, 이는 철골렘, 고양이 등을 포함합니다.
* 구분이 되지 않은 상태의 청크내에서는 동물이 이동하기 때문에 처벌을 받을 수 있습니다.
* 야생에서의 **자동스폰이 아닌 인위적인 동물을 데리고 다닐때에는 5마리까지 허용**됩니다. <mark style="color:green;">ex) 늑대를 데리고 다니는 경우 5마리까지 허용</mark>

🙍🏼‍♀️**🙍🏼‍♂️ 주민**

* **주민은 한청크 안에 5마리**가 가능하며, **청크 반경 9X9 내**에는 위치할수 없습니다.
* 주민이 위치한 청크를 기준으로 상하좌우/대각선 포함 1청크를 띄워야 합니다.
* 반경 **16블록 이내 몹이 5마리 이상 있을 경우 자동 제거됩니다.**

### 『7. 몹공장 및 네더천장』

{% hint style="warning" %}
**다량의 몹 스폰을 유도하여 아이템을 취하는 형태의 몹 공장은 제재 대상이며, 아래 조건을 충족할 시, 몹 공장으로 간주합니다**

* **레드스톤과 호퍼를 사용하거나**, **자동 및 반자동으로 몹을 죽이는 건축물 / 몹타워 / 몹공장/ 몬스터팜 등의 건축물**
* <mark style="color:green;">**스포너 주변 개조 시 6X6 이내 개조가 가능**</mark>하며, <mark style="color:red;">**초과할 경우 공장으로 판정**</mark>합니다.

**공장의 가동 및 이용이 발각되면 **<mark style="color:red;">**경고 2\~3점**</mark>** 부여, 판단이 애매한 경우 관리진의 회의를 거쳐 의견을 수렴합니다.**
{% endhint %}

{% hint style="warning" %}
**몹공장 (개조)**

* **로그확인시 개조한 유저 : 공장위반으로 인한 **<mark style="color:red;">**경고**</mark>**를 부여합니다.**

**몹공장 (킬로그) 악용 위반**

* **개조한 유저와 동일한 마을의 인원의 경우 몹공장 악용으로 인한 판정을 받습니다.**
* **개조한 유저와 관련이 없을 경우 10마리 이상 킬로그가 확인의 될 경우 악용으로 인한 판정을 받습니다.**
{% endhint %}

* 네더 천장의 기반암 평지는 올라갈 수 없으며, 구조물을 건설할 수 없습니다.

### 『8. 도시 / 마을월드』

{% hint style="warning" %}
**해당 규칙을 지키지 않을 시 **<mark style="color:red;">**철거 및 경고**</mark>**를 받을 수 있으며, 부동산의 경계선은 /경계선 명령어로 확인이 가능**합니다.
{% endhint %}

{% hint style="warning" %}
**마을월드는 파밍월드가 아니며**, 마을월드에서 땅을 점유 후 아이템 파밍을 한 뒤에 땅 삭제 시 경고 3\~5점가 부여됩니다
{% endhint %}

**🌳 부동산을 만들 때 필독 !**

* **부동산 생성 시 **<mark style="color:green;">**개인 / 마을 간 부동산의 거리는 최소 48블록(3청크)이상을 유지**</mark>**하여야 합니다.**\
  **(단, 도시월드 및 마을월드 스폰 좌표 +-1500 지역은 제외하며, 인접 부동산이 자신의 부동산이거나 합의된 경우 예외로 합니다.)**
* **청크간 거리 미준수 등으로 신고 처리가 들어온 경우 표지판 부착 및 거리 유지를 요청할 수 있으며, **<mark style="color:green;">**고지를 한 날로부터 3일 이내**</mark>**에 적절한 조치를 취하여야 합니다.**

**🌳 만약, 지역보호가 풀린 부동산을 발견하였다면?**

* **지역보호가 없거나 장기 미접으로 부동산이 사라질 경우, **<mark style="color:green;">**해당 땅의 소유권을 포기한 것으로 간주**</mark>**하며, 부동산 주인은** <mark style="color:green;">**접속 및 지역보호 등의 보호 의무**</mark>**를 지닙니다.**
* **지역보호가 풀린 땅은 서버 소유이며,** <mark style="color:green;">**이용하고자 할 시 신고하기 게시판을 이용하여 부동산 제거 요청 후 사용**</mark>**해야 합니다. ( 미 준수 시, 2항의 약탈 항목을 적용받습니다.)**

### 『10. 거래 및 상점 』

* \*\*유저 상점(/ah) 아이템 시세는 \*\*<mark style="color:green;">**상점 구매가**</mark>**를 초과하여 올릴 수 없습니다.**
* **그림/악보/소설과 같이 타인이 직접 제작한 저작물의 경우,허락을 받지 않은 재 판매가 금지됩니다. (적발시 무통보 회수, 지속시 경고)**

**🧑‍🤝‍🧑아르바이트**

* 아르바이트 진행시에는 서로간의 계약을 지켜야 하며, 예방하기 위한 증빙자료 보관을 권장합니다.
* 아르바이트로 인한 사기 발생 시에는 서버 규칙으로 처벌하며, 관리자가 증빙자료를 요청할 수 있습니다.

🎴 **도박**

* 도박장 운영시에는 **마이너스 배수**를 금합니다.
* 도박사기가 발생되는 경우 서버 법전에 의해 **사기금의 두배를 회수**합니다.
* 도박 확률을 보기 위해 레드스톤을 사용하는 경우 **확률 공개를 요청**할 수 있습니다.
* 복권을 이용한 도박장 운영시에는 **충분한 근거와 추첨 프로그램 공개**로 공정해야 합니다.

### 『11. 처벌』

{% hint style="warning" %}
\*\*인게임/디스코드 신고 접수는 <mark style="color:green;">**\*\*\*\***</mark> \*\*<mark style="color:orange;">**디스코드- 신고하기**</mark>\*\*에서 \*\*<mark style="color:orange;">**신고 접수가 가능**</mark>**합니다**

* 개인 디스코드 및 1:1 문의로 신고접수를 받지 않습니다.
{% endhint %}

* 운영진에게 처벌 강요를 금하고 있습니다.
* ㄴ 운영자는 수많은 업무를 진행하므로, 모든 채팅을 모니터링 할 수 없으며, 일부 누락 된 처벌은 <mark style="color:green;">**디스코드 신고게시판**</mark>을 이용하여야 합니다.
* 잘못된 처벌 결과에 대한 해명은 <mark style="color:green;">**디스코드 해명하기 게시판**</mark>을 이용하여야 하며, 신고 결과에 대한 당사자 외 3자의 개입을 금하고 있습니다.
* 운영진이 판단하기에 서버를 플레이하기 부적합한 유저로 판단 될 시, 처벌을 가합니다.
* ㄴ 타 서버에서의 그릇된 태도, 테러 이력, 본/타 서버의 비신사적 행위, 뒷담화 / 서버 규칙 불이행 / 관리자의 정당한 지시 불복종/서버 시스템 부정
* **게임 내 현금거래/ 후원아이템 거래를 금합니다. (3\~5점의 경고 부여)**
