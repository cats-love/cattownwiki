# 🌍 웹지도

{% hint style="danger" %}
**웹지도는 서비스 준비 중인 기능입니다.**

현재 보안포트 개방을 위한 협력사 문의 진행 중이며, 비활성화 상태입니다.
{% endhint %}

{% hint style="warning" %}
**야생월드, 네더월드, 엔더월드는 웹지도가 제공되지 않습니다.**

해당 월드는 한달 간격으로 초기화되며 청크가 자주 변경되는 월드입니다.
{% endhint %}
