# 🦊 동물 / 몬스터

{% hint style="info" %}
**커스텀 몬스터의 경우, 마인크래프트 공식이 아님을 유의바랍니다.**

🔸 커스텀 몬스터에 대해 고냥이타운은 마인크래프트와 관련이 없습니다.
{% endhint %}

* 기존 야생과 어울리도록 마인크래프트 본연의 야생을 살렸습니다.
* 커스텀 몬스터는 출몰하는 일부 등장하는 바이옴이 다릅니다.
